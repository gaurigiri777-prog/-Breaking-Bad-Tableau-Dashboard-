# 🎬 Breaking Bad Tableau Dashboard | Data Visualization Project

An interactive **Tableau dashboard project** built on the **Breaking Bad web series dataset**, designed to provide storytelling-driven insights into episodes, seasons, IMDB ratings, directors, duration, and U.S. viewership trends.

This project combines **data analytics + visual storytelling + cinematic dashboard design** to create a portfolio-ready Tableau case study.

## 📌 Project Overview

This dashboard provides an in-depth analysis of the **Breaking Bad TV series** using interactive visualizations.

The objective was to transform raw episode-level data into a **storytelling dashboard** that captures the essence of the series while delivering actionable insights.

---

## 🎯 Key Insights Covered

- 📺 Total episodes by season
- ⭐ IMDB ratings across seasons
- 👨‍💼 Directors contribution analysis
- ⏱ Episode duration trends
- 📈 U.S. viewers growth by year
- 🧩 Episode distribution by season and minutes
- 📅 Season / episode level filtering
